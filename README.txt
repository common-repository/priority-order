== Description ==

WooCommerce Priority Order is extension for WooCommerce useful for mark customer orders as priority, sometimes we need mark the orders to differentiate which the order is priority.

== Installation ==

1. Upload `priority-order` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can use button "Priority" at the admin orders page
